C/C++编译

g++编译参数

  • -g 编译带调试信息的可执行文件

  • -o 指定输出文件名

  • -O2 优化源代码

  • -Wall 打印警告信息，-w 关闭警告信息

  • -std=c++11 设置编译标准

  • -D 定义宏

  • -l 指定库文件 -L指定库文件路径

  • -I 指定头文件搜索目录

Cmake

重要指令

  • add_library - 生成库文件

  • add_compile_options - 添加编译参数

  • add_executable - 生成可执行文件

  • add_subdirectory - 向工程添加存放源文件的子目录，其中需有一个CMakeLists.txt

  • include_directories - 引入头文件目录

  • link_directories - 引入库目录，如工程编译时可能用到第三方库的 lib 文件

  • link_libraries - 引入库文件到当前工程（全路径）

  • target_link_libraries - 引入库文件到子工程，即分配，故需指定子工程

  • target_include_directories - 引入头文件目录到子工程，同5

Git

Merge和Rebase

  • git merge 保留了分支的结构与历史提交目录，但提交历史会被大量的 merge 污染

  • git rebase 把所有的提交压缩成一个 patch ，然后把 patch 添加到目标分支里

    优势：项目历史记录线性的，消除不必要的Merge提交

    缺点：丢失了合并提交的上下文：无法看到真实的更改是何时合并到目标分支上的

Linux指令

网络相关命令

  • netstat [选项] 查看网络状态，-i网卡信息，-anp | grep port查看占用端口进程
  • ifconfig 显示或设置网络设备，配置IP、MAC地址、启动网卡
  • fdisk 创建和维护分区表
  • tcpdump 抓包
  • route 查看路由

进程管理命令

  • ps aux 查看当前正在运行的全部进程
  • top 动态显示系统中进程的CPU、物理内存占用
  • free 查看物理内存、交换内存和内核缓冲区内存
  • pstree 以树状图的形式现实系统中的进程
  • kill [选项] pid 默认-15 强制终止-9

文件查找命令

  • sed替换文件内容，sed -i "s/查找字段/替换字段/g" lishan.txt
  • grep输出文件匹配内容，grep 指定字符 源文件
  • more 分⻚查看,通过 cat [文件] | more; f/ 空格下一⻚,回⻋下一行, Q/q 退出
  • less 是 more 的加强版, 可以随意浏览文件,不像more 仅能向前移动
  • head查看前n行，head -n filename，tail后n行
  • which 查找某一命令所在的路径,格式为 which 命令
  • whereis 命令与其类似,但是 whereis 还能展示命令帮助文档所在的路径
  • find 格式为 find 路径 [选项] 关键字 ,常用选项为 -name , -size , -user
  • locate 功能与 find -name 相同,find 是遍历了搜索路径下的每个文件,而 locate 是搜索/var/lib/locatedb 里的一个数据库,相当于散列查找。在使用 locate 命令之前,可先使用 updatedb 命令手动更新数据库。

用户权限命令

  • chmod文件目录权限
    • chmod {augo} {+-=} {rwx} file
    • chmod 754 file，rwx=421
  • chown文件所有者chown usr file
    • chgrp 文件所有组，同
  • useradd [选项] 用户名，-g组 -d目录 -e期限 -u指定ID
    passwd [-d] 用户名，设置/删除密码
    userdel [-r] 用户名，删除用户(及文件)
    usermod 选项 参数 用户名，-l用户名,其余同useradd
  • groupadd [选项] 参数 -g指定用户组ID -r创建系统用户/组
    groupmod 选项 参数 用户组 -g新ID -n修改组名
    groupdel 组名

压缩解压命令

  • zip，解压unzip file.zip
  • tar，压缩-zcvf 解压-zxvf

文件操作命令

  • pwd 查看当前路径。
  • mkdir 创建目录, -p依次创建目录。
  • touch 修改时间属性,也用于创建文件。
  • cp 复制文件,格式 cp [源路径] [目的路径] 。
  • mv 移动文件,格式 mv [源路径] [目的路径] 。
  • rm 删除文件或目录, -r 递归删除, -f 强制。
  • rmdir 删除目录 rmdir [路径] 。
  • wget下载,-c断点续传,-b后台下载并用tail -f wget-log查看进度
  • mount挂载
    • 硬盘:mount 设备 挂载点
    • 镜像:mount -o rw -t iso9660 设备 挂载点,rw读写,ro只读

通配符

通配符含义实例
*匹配任意字符串file_*.txt，匹配file_qseawd.txt等
?匹配一个字符file_?.txt，匹配file_1.txt等
[…]匹配其中一个字符file_[ab].txt，匹配 file_a.txt 和 file_b.txt
[ - ]匹配范围中一个字符file_[a-z].txt，匹配 file_a.txt 至 file_z.txt
[^…]匹配之外的字符file_[^ab].txt，排除 file_a.txt, file_b.txt

管道与重定向

重定向标准输入：<

重定向标准输出： >和>>(追加)

重定向标准错误：2>

重定向同一文件

ls -l /bin/user > ls-output.txt 2>&1
ls -l /bin/user &> ls-output.txt

不处理输出

ls -l /bin/user 2> /dev/null

管道：| ，将一个命令的标准输出传送到另一个命令的标准输入中。

ls -l /user/bin | less
ls /bin /user/bin | grep zip

屏幕级yH从屏幕首行开始复制直到当前行(包括)
屏幕级yL从当前行开始复制直到屏幕尾行(包括)
文档级yG从当前行开始复制直到文档尾行(包括)
文档级ynG从第n行开始复制直到当前行(包括)
