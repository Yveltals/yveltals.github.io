Jdk

  1. 在oracle官网 下载deb

  2. 安装后会在目录/usr/lib/jvm下生成jdk 文件

  3. 修改环境变量: sudo vim ~/.bashrc

    使环境变量马上生效：source ~/.bashrc

  4. 系统注册此jdk (可选?)

  5. 查看java版本，看看是否安装成功：java -version

Mysql

最新

1. 命令安装

通过apt安装的mysql已经默认开启服务，服务名叫做mysql，而非mysqld

2. 登陆Mysql

方法一：sudo mysql

方法二：查看默认用户密码登陆

3. 本地Root用户

  • 修改一个host字段为 localhost 的 root 用户密码（localhost 表示本机登录）
  • 不需授予访问权限，因为默认已有

4. 远程Root用户

  • 创建一个 host 字段为 % 的 root 用户（% 表示远程登陆）
  • 授权所有数据库的访问权限
  • 开启远程连接，注释bind-address

5. 查看用户信息

Ubuntu修改主机名注意事项

修改主机名后，要把etc/hosts文件中主机名更新，否则springboot启动日志获取不到主机名，导致项目启动缓慢

显示主机名：hostname
显示主机IP：hostname -i
临时设置主机名：hostname test100

去APK弹窗

针对部分破解软件的开屏提示，强迫症自救

去弹窗

  1. MT管理器查看apk，用dex编辑器++打开dex文件(全选)
  2. 搜索内容show，搜索类型代码 （或搜弹窗内容等）
  3. 找到类似invoke-virtual{v0},LandroidlapplAlertDialog$Builder;->show()Landroidl...
  4. 点击跳转到相应代码，用#注释该句

去浮窗

  1. 同理搜索相应浮窗内容，搜索类型字符串
  2. 跳转将对应方法注释
