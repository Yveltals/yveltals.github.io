Redis

使用 C++ SDK Hiredis 访问 redis
Redis Stream

1. 下载源码解压
wget http://download.redis.io/releases/redis-5.0.5.tar.gz
tar -xzf redis-5.0.5.tar.gz && cd redis-5.0.5
2. 编译
yum -y install gcc gcc-c++ kernel-devel
make
3. 安装
make PREFIX=/usr/local/redis install
cp redis.conf /usr/local/redis/bin/
4. 更改配置
vim /usr/local/redis/bin/redis.conf

5. 配置环境变量
vim /etc/profile
export PATH=”$PATH:/usr/local/redis/bin”
source /etc/profile
6. 启动脚本

./redis start|stop|restart

  • 需求：按ctrl+tab切换最近两个标签页

Ubuntu相关

1. 无法访问部分网页

问题描述：双系统win可以，但Ubuntu很多网页如CSDN无法访问

解决方案：Ubuntu网卡的MTU设置不当，改为1412（尝试值）即可

2. 开关机异常缓慢

  • 开机动画
  • 关机重启等待 90 秒

3. 双系统扩容
预留给 Ubuntu 系统的预留空间不足时需要扩容，可 参考流程 与 gparted使用方法

  • 启动盘进入系统，使用自带的gparted分区工具
  • 将多余空间分区并格式化
  • 通过一系列对未分配分区的调换，移到待扩容分区后方相邻合并

4. Linux开机挂载硬盘

  • 通过sudo blkid查看C盘UUID，如C8086A73086A6104
  • 挂载目录如/media/yveltal/LENOVO
  • 文件系统类型，如win为ntfs

5. 通过tar包安装应用

  1. 解压下载文件并移动到安装文件夹内/opt/demo

  2. 为启动文件建立软Link(注意修改.sh中路径)

  1. 添加第三方软件图标 sudo vim /usr/share/applications/demo.desktop
