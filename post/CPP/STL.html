关联容器

set / map

set: 去重并排序，基于树

multiset: 允许重复 key 值的 set

unordered_set: 只去重不排序，基于哈希表，增删查O(1)

map: 自动按Key升序排序（红黑树），增删查改O(logn)

unorder_map: 哈希存储无序，增删查改O(1)
