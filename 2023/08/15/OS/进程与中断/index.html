иҝӣзЁӢдёҺзәҝзЁӢзҡ„еҢәеҲ«

иҖҒе…«иӮЎзүҲжң¬

  • дёҖдёӘзәҝзЁӢеұһдәҺдёҖдёӘиҝӣзЁӢпјҢдёҖдёӘиҝӣзЁӢеҢ…еҗ«еӨҡдёӘзәҝзЁӢ
  • иҝӣзЁӢжҳҜиө„жәҗеҲҶй…Қзҡ„еҹәжң¬еҚ•дҪҚ
    • жңүзӢ¬з«Ӣзҡ„ең°еқҖз©әй—ҙпјҢжӢҘжңүCPUиө„жәҗгҖҒеҶ…еӯҳиө„жәҗгҖҒж–Үд»¶иө„жәҗзӯү
  • зәҝзЁӢжҳҜCPUи°ғеәҰзҡ„еҹәжң¬еҚ•дҪҚ
    • з§ҒжңүпјҲзәҝзЁӢдёҠдёӢж–ҮпјүпјҡзәҝзЁӢж ҲпјҲдҪҶиғҪйҖҡиҝҮжҢҮй’Ҳиў«еҸҰдёҖдёӘзәҝзЁӢдҝ®ж”№пјүпјҢеҜ„еӯҳеҷЁпјҢзЁӢеәҸи®Ўж•°еҷЁ
    • е…ұдә«пјҡиҝӣзЁӢең°еқҖз©әй—ҙдёӯйҷӨзәҝзЁӢдёҠдёӢж–Үд№ӢеӨ–зҡ„жүҖжңүеҶ…е®№пјҡе ҶгҖҒе…ЁеұҖеҸҳйҮҸгҖҒйқҷжҖҒеҸҳйҮҸгҖҒд»Јз Ғж®өгҖҒж–Үд»¶зӯүе…¬е…ұиө„жәҗ
  • ж•°жҚ®е…ұдә«пјҡиҝӣзЁӢжңүзӢ¬з«Ӣзҡ„ең°еқҖз©әй—ҙпјҢж•°жҚ®йҡ”зҰ»пјҢе…ұдә«еӨҚжқӮпјҢйңҖиҰҒIPCпјӣзәҝзЁӢе…ұдә«иҝӣзЁӢж•°жҚ®
  • дёҠдёӢж–ҮеҲҮжҚўпјҡиҝӣзЁӢеҲҮжҚўйңҖиҰҒеҲ·ж–°TLBпјҲеҜ№дәҺдёҚеҗҢиҝӣзЁӢпјҢзӣёеҗҢиҷҡжӢҹең°еқҖдјҡжҳ е°„дёҚеҗҢзү©зҗҶең°еқҖдә§з”ҹжӯ§д№үпјүе№¶иҺ·еҸ–ж–°зҡ„ең°еқҖз©әй—ҙгҖҒз„¶еҗҺеҲҮжҚўзЎ¬д»¶дёҠдёӢж–Үе’ҢеҶ…ж ёж ҲпјӣзәҝзЁӢеҲҮжҚўж—¶еҸӘйңҖиҰҒеҲҮжҚўзЎ¬д»¶дёҠдёӢж–Үе’ҢеҶ…ж ёж Ҳ
  • еҸҜйқ жҖ§пјҡеӨҡиҝӣзЁӢжӣҙеҸҜйқ пјҢдёҖдёӘиҝӣзЁӢеҙ©жәғеҗҺпјҢеңЁдҝқжҠӨжЁЎејҸдёӢдёҚдјҡеҜ№е…¶д»–иҝӣзЁӢдә§з”ҹеҪұе“ҚпјӣдҪҶзәҝзЁӢд№Ӣй—ҙжІЎжңүйҡ”зҰ»жҖ§пјҢдёҖдёӘзәҝзЁӢеҙ©жәғж—¶еҸҜиғҪе·Із»Ҹз ҙеқҸдәҶе…¶д»–зәҝзЁӢзҡ„еҶ…еӯҳпјҢеҜјиҮҙж•ҙдёӘиҝӣзЁӢkill
  • йҖҡдҝЎж–№ејҸпјҡзәҝзЁӢе…ұдә«иҝӣзЁӢж•°жҚ®пјҢзәҝзЁӢйҖҡдҝЎдё»иҰҒз”ЁдәҺзәҝзЁӢеҗҢжӯҘпјҢжүҖд»ҘжІЎжңүIPCж•°жҚ®дәӨжҚўзҡ„жңәеҲ¶

еҶ…ж ёдёӯеҜ№зәҝзЁӢзҡ„иЎЁзӨә

еңЁ Linux дёӯпјҢж— и®әиҝӣзЁӢиҝҳжҳҜзәҝзЁӢпјҢйғҪжҳҜжҠҪиұЎжҲҗдәҶ task д»»еҠЎпјҢеңЁжәҗз ҒйҮҢйғҪжҳҜз”Ё task_struct з»“жһ„жқҘе®һзҺ°зҡ„гҖӮ

еҜ№дәҺзәҝзЁӢжқҘи®ІпјҢжүҖжңүзҡ„еӯ—ж®өйғҪжҳҜе’ҢиҝӣзЁӢдёҖж ·зҡ„пјҲжң¬жқҘе°ұжҳҜдёҖдёӘз»“жһ„дҪ“пјүгҖӮеҢ…жӢ¬зҠ¶жҖҒгҖҒpidгҖҒtask ж ‘е…ізі»гҖҒең°еқҖз©әй—ҙгҖҒж–Үд»¶зі»з»ҹдҝЎжҒҜгҖҒжү“ејҖзҡ„ж–Үд»¶дҝЎжҒҜзӯүзӯүеӯ—ж®өпјҢзәҝзЁӢд№ҹйғҪжңүгҖӮ

еңЁ Linux дёӯпјҢжҜҸдёҖдёӘ task_struct йғҪйңҖиҰҒиў«е”ҜдёҖзҡ„ж ҮиҜҶпјҢе®ғзҡ„ pid е°ұжҳҜе”ҜдёҖж ҮиҜҶеҸ·пјҢеҚіжҜҸдёӘиҝӣзЁӢ/зәҝзЁӢзҡ„pidйғҪдёҚеҗҢгҖӮеҜ№дәҺзәҝзЁӢпјҢиҝҳйңҖйҖҡиҝҮ tgid еӯ—ж®өжқҘиЎЁзӨәе…¶жүҖеҪ’еұһзҡ„иҝӣзЁӢ IDгҖӮ

зәҝзЁӢеҲӣе»әиҝҮзЁӢ

еҸҜи§ҒдёӨиҖ…еҲӣе»әиҝҮзЁӢе·®дёҚеӨҡпјҢдёҖж ·дҪҝз”Ёзҡ„жҳҜеҶ…ж ёйҮҢзҡ„ do_fork еҮҪж•°пјҢжңҖеҗҺиө°еҲ° copy_process жқҘе®Ңж•ҙеҲӣе»әгҖӮcopy_process е…ҲжҳҜеӨҚеҲ¶дәҶдёҖдёӘж–°зҡ„ task_struct еҮәжқҘпјҢз„¶еҗҺеҜ№ task_struct дёӯзҡ„еҗ„з§Қж ёеҝғеҜ№иұЎиҝӣиЎҢжӢ·иҙқеӨ„зҗҶпјҢиҝҳз”іиҜ·дәҶ pid гҖӮ

еҢәеҲ«еңЁдәҺдәҢиҖ…еңЁи°ғз”Ё do_fork ж—¶дј е…Ҙзҡ„ clone_flags йҮҢзҡ„ж Үи®°дёҚдёҖж ·пјҡ

  • еҲӣе»әиҝӣзЁӢж—¶зҡ„ flagпјҡд»…жңүдёҖдёӘ SIGCHLD
  • еҲӣе»әзәҝзЁӢж—¶зҡ„ flagпјҡеҢ…жӢ¬ CLONE_VMгҖҒCLONE_FSгҖҒCLONE_FILESгҖҒCLONE_SIGNALгҖҒCLONE_SETTLSгҖҒCLONE_PARENT_SETTIDгҖҒCLONE_CHILD_CLEARTIDгҖҒCLONE_SYSVSEMгҖӮ

еҜ№дәҺзәҝзЁӢжқҘи®ІпјҢз”ұдәҺдј е…ҘдәҶflagsпјҢе…¶ең°еқҖз©әй—ҙ mm_structгҖҒзӣ®еҪ•дҝЎжҒҜ fs_structгҖҒжү“ејҖж–Үд»¶еҲ—иЎЁ files_struct йғҪжҳҜе’ҢеҲӣе»әе®ғзҡ„д»»еҠЎе…ұдә«зҡ„пјҢж— йңҖеҲӣе»әгҖӮ

еҜ№дәҺиҝӣзЁӢжқҘи®ІпјҢжІЎдј  flagsпјҢең°еқҖз©әй—ҙ mm_structгҖҒжҢӮиҪҪзӮ№ fs_structгҖҒжү“ејҖж–Үд»¶еҲ—иЎЁ files_struct йғҪиҰҒжҳҜзӢ¬з«ӢжӢҘжңүзҡ„пјҢйғҪйңҖиҰҒеҺ»з”іиҜ·еҶ…еӯҳе№¶еҲқе§ӢеҢ–е®ғд»¬гҖӮ

жҖ»з»“

еӣ дёәеңЁеҶ…ж ёдёӯзәҝзЁӢе’ҢиҝӣзЁӢйғҪжҳҜз”Ё task_struct жқҘиЎЁзӨәпјҢеҸӘдёҚиҝҮзәҝзЁӢе’ҢиҝӣзЁӢзҡ„еҢәеҲ«жҳҜдјҡе’ҢеҲӣе»әе®ғзҡ„зҲ¶иҝӣзЁӢе…ұдә«жү“ејҖж–Үд»¶еҲ—иЎЁгҖҒзӣ®еҪ•дҝЎжҒҜгҖҒиҷҡжӢҹең°еқҖз©әй—ҙзӯүж•°жҚ®з»“жһ„пјҢдјҡжӣҙиҪ»йҮҸдёҖдәӣпјҢжүҖд»ҘеңЁ Linux дёӢзҡ„зәҝзЁӢд№ҹеҸ«иҪ»йҮҸзә§иҝӣзЁӢгҖӮ

еҜ№дәҺеҶ…ж ёд»»еҠЎжқҘиҜҙпјҢж— и®әжңүеӨҡе°‘дёӘд»»еҠЎпјҢе…¶дҪҝз”Ёең°еқҖз©әй—ҙйғҪжҳҜеҗҢдёҖдёӘгҖӮжүҖд»ҘдёҖиҲ¬йғҪеҸ«еҶ…ж ёзәҝзЁӢпјҢиҖҢдёҚжҳҜеҶ…ж ёиҝӣзЁӢгҖӮ

CPUж ёеҝғгҖҒзәҝзЁӢж•°жҰӮеҝө

CPUзү©зҗҶж•°

cat /proc/cpuinfo | grep 'physical id' | sort | uniq | wc -l

зү©зҗҶCPUж•°йҮҸпјҢжҷ®йҖҡз”өи„‘дёҖиҲ¬еҸӘжңүдёҖдёӘCPUжҸ’ж§ҪпјҢд№ҹе°ұжҳҜеҸӘжңүдёҖдёӘзү©зҗҶCPU

COREж ёеҝғж•°(зү©зҗҶж ёеҝғж•°)

cat /proc/cpuinfo | grep 'core id' | sort | uniq | wc -l

дёҖдёӘзү©зҗҶCPUжӢҘжңүзҡ„иҝҗз®—еҚ•е…ғдёӘж•°пјҢжҜҸдёӘиҝҗз®—еҚ•е…ғжңүдёҖеҘ—еҜ„еӯҳеҷЁгҖҒL1гҖҒL2з§Ғжңүзј“еӯҳгҖӮдёҖдёӘж ёеҝғеҗҢдёҖж—¶еҲ»еҸӘиғҪиҝҗиЎҢдёҖдёӘзәҝзЁӢпјҢдёҚи®әиҝҷдёӘзәҝзЁӢжҳҜе“ӘдёӘиҝӣзЁӢзҡ„гҖӮжҜҸйҡ”дёҖе®ҡж—¶й—ҙе°ұдјҡеҲҮжҚўзәҝзЁӢпјҢеҰӮжһңеҲҮжҚўеҲ°е…¶е®ғиҝӣзЁӢзҡ„зәҝзЁӢпјҢдјҡеҲҮжҚўCR3пјҲиҝӣзЁӢеҲҮжҚўж—¶пјҢдјҡе°ҶиҜҘиҝӣзЁӢйЎөиЎЁзҡ„жҢҮй’ҲеҠ иҪҪеҲ°CR3еҜ„еӯҳеҷЁпјүгҖӮ

THREADзәҝзЁӢж•°(йҖ»иҫ‘ж ёеҝғж•°)

cat /proc/cpuinfo | grep 'processor id' | sort | uniq | wc -l

иӢұзү№е°”зҡ„и¶…зәҝзЁӢжҠҖжңҜпјҢеҸҜд»Ҙе®һзҺ°дёҖдёӘж ёеҝғеҗҢж—¶жү§иЎҢеӨҡдёӘзәҝзЁӢпјҢеҚійҖ»иҫ‘ж ёеҝғж•°зҡ„иҜҙжі•пјҡ8ж ё16зәҝзЁӢпјҢдёҖдёӘж ёеҝғиғҪи·‘дәҢдёӘзәҝзЁӢпјҢ

CPUеңЁжү§иЎҢдёҖжқЎжңәеҷЁжҢҮд»Өж—¶пјҢе№¶дёҚдјҡе®Ңе…Ёең°еҲ©з”ЁжүҖжңүзҡ„CPUиө„жәҗпјҢжҳҜжңүеӨ§йҮҸиө„жәҗиў«й—ІзҪ®зқҖзҡ„гҖӮи¶…зәҝзЁӢжҠҖжңҜе…Ғи®ёдёӨдёӘзәҝзЁӢеҗҢж—¶дёҚеҶІзӘҒең°дҪҝз”ЁCPUдёӯзҡ„иө„жәҗгҖӮжҜ”еҰӮдёҖжқЎж•ҙж•°иҝҗз®—жҢҮд»ӨеҸӘдјҡз”ЁеҲ°ж•ҙж•°иҝҗз®—еҚ•е…ғпјҢжӯӨж—¶жө®зӮ№иҝҗз®—еҚ•е…ғе°ұз©әй—ІдәҶпјҢиӢҘдҪҝз”ЁдәҶи¶…зәҝзЁӢжҠҖжңҜпјҢдё”еҸҰдёҖдёӘзәҝзЁӢеҲҡеҘҪжӯӨж—¶иҰҒжү§иЎҢдёҖдёӘжө®зӮ№иҝҗз®—жҢҮд»ӨпјҢCPUе°ұе…Ғи®ёеұһдәҺдёӨдёӘдёҚеҗҢзәҝзЁӢзҡ„ж•ҙж•°иҝҗз®—жҢҮд»Өе’Ңжө®зӮ№иҝҗз®—жҢҮд»ӨеҗҢж—¶жү§иЎҢпјҢиҝҷжҳҜзңҹзҡ„е№¶иЎҢгҖӮ

и¶…зәҝзЁӢвҖқеҸҜиғҪвҖқжҳҜе№¶иЎҢзҡ„,д№ҹеҸҜиғҪдёҚжҳҜе№¶иЎҢзҡ„пјҢдҪҶиҝҷд№ҹе№¶дёҚж„Ҹе‘ізқҖдёӨдёӘзәҝзЁӢеңЁеҗҢдёҖдёӘCPU(д№ҹе°ұжҳҜеҗҢдёҖдёӘж ёеҝғ)дёӯдёҖзӣҙйғҪеҸҜд»Ҙе№¶иЎҢжү§иЎҢпјҢеҸӘжҳҜжҒ°еҘҪзў°еҲ°дёӨдёӘзәҝзЁӢеҪ“еүҚиҰҒжү§иЎҢзҡ„жҢҮд»ӨдёҚдҪҝз”ЁзӣёеҗҢзҡ„CPUиө„жәҗж—¶жүҚеҸҜд»ҘзңҹжӯЈең°е№¶иЎҢжү§иЎҢгҖӮ

иҝӣзЁӢзҡ„еӨҡдёӘзәҝзЁӢиғҪеӨҡж ёе№¶иЎҢеҗ—

еҶ…ж ёжҖҒзәҝзЁӢ

  • дјҳзӮ№пјҡдёҖдёӘиҝӣзЁӢзҡ„еӨҡдёӘзәҝзЁӢеҸҜд»ҘеҲҶй…ҚеҲ°еӨҡдёӘCOREж ёеҝғеҗҢж—¶жү§иЎҢпјҢе№¶дё”з”ұдәҺеҶ…ж ёзә§зәҝзЁӢеҸӘжңүеҫҲе°Ҹзҡ„ж•°жҚ®з»“жһ„е’Ңе Ҷж ҲпјҢеҲҮжҚўйҖҹеәҰеҝ«пјҢжң¬иә«д№ҹеҸҜд»Ҙз”ЁеӨҡзәҝзЁӢжҠҖжңҜе®һзҺ°жҸҗй«ҳзі»з»ҹзҡ„иҝҗиЎҢйҖҹзҺҮгҖӮ
  • зјәзӮ№пјҡзәҝзЁӢеңЁз”ЁжҲ·жҖҒзҡ„иҝҗиЎҢпјҢиҖҢзәҝзЁӢзҡ„и°ғеәҰе’Ңз®ЎзҗҶеңЁеҶ…ж ёе®һзҺ°пјҢеңЁжҺ§еҲ¶жқғд»ҺдёҖдёӘзәҝзЁӢдј йҖҒеҲ°еҸҰдёҖдёӘзәҝзЁӢйңҖиҰҒз”ЁжҲ·жҖҒеҲ°еҶ…ж ёжҖҒеҶҚеҲ°з”ЁжҲ·жҖҒзҡ„жЁЎејҸеҲҮжҚўпјҢжҜ”иҫғеҚ з”Ёзі»з»ҹиө„жәҗгҖӮ

з”ЁжҲ·жҖҒзәҝзЁӢ

  • дјҳзӮ№пјҡзәҝзЁӢзҡ„и°ғеәҰдёҚйңҖиҰҒеҶ…ж ёзӣҙжҺҘеҸӮдёҺпјҢжҺ§еҲ¶з®ҖеҚ•пјҢеҸҜд»ҘеңЁдёҚж”ҜжҢҒзәҝзЁӢзҡ„ж“ҚдҪңзі»з»ҹдёӯе®һзҺ°
  • зјәзӮ№пјҡжІЎжңүеҠһжі•дҪҝз”ЁеӨҡж ёеҝғе№¶иЎҢжү§иЎҢ
